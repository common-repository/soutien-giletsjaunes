Soutiens aux gilets jaunes en plaçant un ruban indiquant "#GiletsJaunes" sur votre site.

== Description ==

Un ruban avec les mots "#GiletsJaunes" sera placé dans le coin supérieur droit ou gauche de votre site Web, renvoyant au groupe Facebook https://www.facebook.com/groups/357767044781992/. Regardez les captures d'écran pour voir à quoi ressemble le ruban. Merci de votre soutien.

== Installation ==

1. Téléchargez l’archive zip du plugin.
2. Extrayez-le dans votre dossier "wp-content/plugins" ou téléchargez-le via le panneau d'administration.
3. Naviguez jusqu’à la section de vos plugins et activez le plugin dont vous avez besoin (côté gauche ou droit, mais pas les deux).
